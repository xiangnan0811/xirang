# Contributor Covenant Code of Conduct

## 我们的承诺

为了营造开放和友好的环境，我们作为贡献者和维护者承诺，无论年龄、体型、可见或不可见的残疾、种族、性征、性别认同与表达、经验水平、教育程度、社会与经济地位、国籍、个人外貌、人种、种姓、肤色、宗教信仰、性别取向或性向，参与本项目和我们的社区都将是无骚扰的体验。

我们承诺以有助于建设开放、热情、多元、包容和健康社区的方式行事。

## 我们的标准

有助于营造积极环境的行为示例：

- 表现出同理心和善意
- 尊重不同的意见、观点和经验
- 提供和接受建设性反馈
- 承担责任并向受我们错误影响的人道歉，并从经验中学习
- 关注的不仅仅是个人利益，更是整个社区的整体利益

不可接受的行为示例：

- 使用色情化的语言或图像，以及任何形式的性关注或挑逗
- 挑衅、侮辱或贬损性评论，以及个人或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息（如住址、电邮地址等）
- 在专业场合可被合理认定为不当的其他行为

## 执行责任

社区维护者负责澄清和执行可接受行为的标准，并将对其认为不当、威胁、冒犯或有害的任何行为采取适当且公正的纠正措施。

社区维护者有权移除、编辑或拒绝与本行为准则不一致的评论、提交、代码、wiki 编辑、issue 和其他贡献，并在适当时说明理由。

## 适用范围

本行为准则适用于所有社区场合，也适用于个人在公开场合代表社区时。代表本社区的例子包括：使用官方电子邮件地址、通过官方社交媒体账号发布信息、作为指定代表参加线上或线下活动等。

## 执行

可通过项目维护者邮箱报告辱骂、骚扰或其他不可接受的行为：详见 [`SECURITY.md`](SECURITY.md) 中提供的联系方式。所有投诉将被及时和公正地审查与调查。

社区维护者有义务尊重报告者的隐私和安全。

## 执行准则

社区维护者将在判定违反本行为准则的行为后果时遵循以下社区影响准则：

### 1. 纠正

**社区影响**：使用了不当语言或其他在社区中被视为不专业或不受欢迎的行为。

**后果**：来自社区维护者的私下书面警告，提供违规说明并解释为何该行为不当。可能要求公开道歉。

### 2. 警告

**社区影响**：单一事件或一系列行为构成的违规。

**后果**：警告并附以持续行为后果。在指定时间内不得与相关人员有任何互动，包括对执行行为准则者的非主动互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定时间内禁止与社区进行任何形式的互动或公开沟通。在此期间不允许与相关人员有任何公开或私下的互动，包括对执行行为准则者的非主动互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出违反社区标准的模式，包括持续的不当行为、骚扰个人或攻击/贬低某类人群。

**后果**：永久禁止参与社区内任何形式的公开互动。
