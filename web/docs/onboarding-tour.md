# 新手引导功能

## 概述

新手引导系统提供交互式分步引导，帮助新用户快速上手息壤平台。

## 功能特性

### 1. 自动触发
- 首次登录时自动显示引导对话框
- 基于 localStorage 持久化引导状态

### 2. 分步引导
引导流程包含 4 个关键步骤：
1. **配置 SSH Key** - 添加可复用的 SSH 密钥
2. **添加节点** - 接入至少一个服务器节点
3. **创建策略** - 定义 rsync 备份规则
4. **创建任务** - 绑定节点与策略并触发执行

### 3. 智能推进
- 自动检测步骤完成状态
- 完成当前步骤后自动推进到下一步
- 实时显示进度百分比

### 4. 灵活控制
- **跳过引导** - 用户可随时跳过引导流程
- **重新开始** - 完成或跳过后可通过右下角按钮重新开始
- **继续引导** - 引导进行中时可最小化，稍后继续

### 5. 视觉反馈
- 弹窗式对话框展示当前步骤
- 进度条显示整体完成度
- 步骤卡片高亮当前进度
- 浮动提示器显示引导状态

## 技术实现

### 组件位置
- 主组件：`/web/src/components/onboarding-tour.tsx`
- 测试文件：`/web/src/components/onboarding-tour.test.tsx`

### 集成方式
在 `app-shell.tsx` 中全局集成，通过 props 传递数据计数：

```tsx
<OnboardingTour
  sshKeysCount={consoleData.sshKeys.length}
  nodesCount={consoleData.nodes.length}
  policiesCount={consoleData.policies.length}
  tasksCount={consoleData.tasks.length}
/>
```

### 状态管理
使用 `usePersistentState` hook 持久化引导状态：
- `completed`: 是否已完成所有步骤
- `currentStepIndex`: 当前步骤索引
- `dismissed`: 是否已跳过引导

### 自动推进逻辑
通过 `useEffect` 监听数据计数变化，自动检测步骤完成状态并推进。

## 用户体验

### 首次访问流程
1. 用户首次登录后 500ms 自动弹出欢迎对话框
2. 点击"开始引导"跳转到 SSH Key 配置页
3. 完成配置后自动弹出下一步引导
4. 依次完成所有步骤

### 中断与恢复
- 用户可随时关闭对话框，右下角显示浮动进度提示
- 点击浮动提示可重新打开引导对话框
- 引导状态持久化，刷新页面后继续

### 跳过与重置
- 点击"跳过引导"永久关闭引导流程
- 右下角显示"重新引导"按钮，可随时重新开始
